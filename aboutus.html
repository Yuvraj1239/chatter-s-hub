inspiration of this application

Many people say conversational UI is the future of web interface. Some might say it`s just another design fad or that text-based interfaces aren`t anything new with the advent of iPhone Messages, Slack or Facebook Messenger the way we exchange information changed irreversibly. Text messages have became extremely natural way of communicating these days. So in this post we`ve gathered 40 Inspiring Chat UI Designs for inspiration that you can use to further the design of chat UI elements to create great experiences with users.
